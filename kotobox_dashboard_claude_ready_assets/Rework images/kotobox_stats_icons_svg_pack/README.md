# Kotobox stats icons SVG pack

This pack contains the SVG icons matching the stat cards shown in your reference:
- Study Streak
- Studied Today
- Reviews Due
- Weekly Goal

Folders:
- `01_small_icons/` → the small label icons used at the top-left of each card
- `02_large_ring_icons/` → the larger circular icons used on the first three cards
- `stats-icons-sheet.svg` → combined preview sheet

All files are transparent SVGs and can be dropped directly into Claude/project assets.
