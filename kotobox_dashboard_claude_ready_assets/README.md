## Use these first

1. `00_reference/dashboard-overview-reference.png`
   - Full dashboard reference.

2. `01_mascots_png_transparent/`
   - Transparent mascot PNGs.
   - These are the mascot files Claude should use.

3. `02_svg_icons/`
   - SVG icons for navigation, dashboard utility icons, categories, and badges.

4. `03_png_exports/`
   - Transparent PNG exports of the SVG icons.

5. `04_backgrounds/night-landscape.svg`
   - Background illustration.

6. `05_exact_png_crops/`
   - Exact screenshot crops for visual reference.

7. `assets-preview.html`
   - Open this to preview every asset.

8. `06_prompt/claude-dashboard-asset-prompt.txt`
   - Paste this into Claude.

## Important

The mascot is provided as transparent PNG because the exact mascot from the reference is raster artwork, not a true SVG vector.
The icons are available as SVG and transparent PNG exports.
