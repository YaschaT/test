Kotobox Lens.

Snip Japanese text anywhere on screen and get its reading, meaning, kanji breakdown, and an example sentence. Powered by your own Anthropic API key, which is stored only on this device. Each lookup costs a fraction of a cent.

How to use: click the Kotobox Lens icon in the toolbar (or press Alt+Shift+K), then drag a box around any Japanese text (a subtitle, a menu, a tweet, a manga panel). Esc cancels.
